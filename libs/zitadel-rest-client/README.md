# zitadel-rest-client

Thin async REST client for the Zitadel Management API v2. Consumed by
`pim-bootstrap` to drive project/app/user/role provisioning. Not intended as
a general-purpose Zitadel SDK.

See `docs/decisions/0016-rest-client-for-zitadel-management-api.md` for the
architectural rationale and endpoint inventory.
